# Customize a duplicated Report Builder template

After duplicating a template, you can add or remove columns, update filters, rename columns, and adjust sorting before saving and downloading\. This article covers each customization step\.

Before you start, duplicate a template from the **Templates** tab.

## Add and remove columns

1. In the edit view, locate the data set that contains the column you want to add.
2. Select **+** next to the column name. The column appears in the report canvas.
3. To add the same column more than once, for example, to count two different status values. Select **+** again for that column.
4. To remove a column, select it in the canvas, then select the remove icon.

## Reorder and rename columns

1. Drag a column to the desired position in the canvas. The order in the canvas matches the column order in the downloaded file.
2. To rename a column, select its alias field and enter the name you want to appear as the column header in the report.

## Update filters

1. To change an existing filter value, select the filter row, edit the value, then confirm.
2. To add a new filter, select **Add filter**.
3. Choose the field you want to filter on.
4. Select an operator. Available operators depend on the field's data type.
5. Enter or select a value:
    * For string fields, type to search and select from the suggestions.
    * For list fields, select one or more values from the picker.
    * For date fields, enter a date or choose a relative range such as last 90 days.
6. To remove a filter, select the delete icon on that filter row.

## Update sorting

1. Select the sort icon on the column you want to sort by.
2. Select **Ascending** or **Descending**.
3. To sort on additional columns, repeat for each column. The first sorted column is primary; additional sorts apply within ties.

>[!TIP]
>
>Always apply at least one sort. Without sorting, row order may differ between downloads of the same report.

## Save and download

1. Select Save to save your changes to the **Reports** tab.
2. Select **Download** to generate the report. You'll receive an in-app notification when the file is ready.
