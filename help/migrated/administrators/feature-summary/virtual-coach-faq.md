# Virtual Coach - Frequently asked questions 

**What is Virtual Coach?**

Virtual Coach is an AI-powered roleplay and assessment feature built into Adobe Learning Manager. It lets learners practice real-world conversations with an AI persona that responds intelligently in real time. After each session, learners receive an instant performance report covering what they said and how they said it.

**Who can use Virtual Coach?**

Virtual Coach is available to all learners, authors, and administrators in an Adobe Learning Manager account where Virtual Coach has been activated. Learners access and complete roleplay sessions. Authors create and publish roleplay scenarios. Administrators manage credits and view reports.

**What types of roleplays are available?**

Virtual Coach supports three domains: Sales Enablement, Leadership Development, and Skills Assessment. Authors choose from more than 50 pre-built templates covering scenarios such as B2B discovery calls, cold calls, handling objections, giving difficult feedback, and de-escalating customer complaints. Authors can also create custom scenarios from scratch using the AI assistant.

**What languages does Virtual Coach support?**

Virtual Coach is available in 16 languages for both the interface and simulation content, including English, French, German, Spanish, Portuguese, Japanese, Chinese (Mandarin), Dutch, Polish, Italian, Korean, Hindi, Turkish, Malay, Hungarian, Danish, Arabic, Swedish, Norwegian, Czech, and Greek.

**How is Virtual Coach licensed and billed?**

Virtual Coach is available as an add-on subscription to Adobe Learning Manager. Usage is measured in Monthly Active Users (MAUs). An MAU credit is consumed when a learner completes a session in a calendar month. Additional sessions by the same learner in that month do not consume additional credits. Unused credits at the end of the annual contract lapse.

**How is the learner's score calculated?**

Each session produces a knowledge score and a style score. The knowledge score reflects whether the learner addressed the required topics and provided accurate information. The style score reflects how the learner communicated, including pace, clarity, filler words, sentence strength, and vocal energy. The authors set the weight of each component when configuring the scenario. A common configuration is 70% knowledge and 30% style.

**Can a learner retry a roleplay?**

Yes. Learners can attempt a roleplay as many times as they want. Each attempt is a new independent session and generates a new performance report. Only the first session in a calendar month consumes an MAU credit.

**Can learners submit their sessions for human review?**

Yes, if the author enables this option during scenario setup. When enabled, learners can submit their session recording for review by a designated evaluator. The author configures this in the Score Settings section of the roleplay editor.

**Is Virtual Coach available on mobile?**

Virtual Coach is supported on the Adobe Learning Manager desktop and mobile web and via APIs. It is not available in the Adobe Learning Manager mobile app (iOS/Android) in the current release.

**Is learner data used to train the AI?**

No. Virtual Coach is hosted on a GDPR-compliant infrastructure, and no personal learner data is used to train AI models.

**What is the difference between a job aid and a course module for Virtual Coach?**

A job aid is a standalone, on-demand resource that learners can access directly from the Catalog at any time without being enrolled in a course. A course module is accessed as part of a structured course sequence with enrollment, completion tracking, and formal assessment. The same roleplay can be published as a job aid and added to multiple courses simultaneously.

**Is Virtual Coach supported on headless or API implementations?**

Virtual Coach is not currently supported on AEM Sites, AEM Widgets, or custom headless implementations. This may be included in a future release.
