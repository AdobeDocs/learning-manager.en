# Report Builder in Adobe Learning Manager

Virtual Coach in Adobe Learning Manager is an AI-powered learning play and assessment feature that lets learners practice real-world conversations through voice or avatar-based roleplay chat with a responsive AI persona. It simulates realistic scenarios such as sales or leadership situations, adapting in real time and providing instant, detailed feedback on both content and delivery. Like a flight simulator for communication and skills training, it enables employees to build confidence and competence by practicing complex interactions before facing them in real-life workplace situations.

As an author, you have an important role to create realistic roleplays using the Virtual Coach incorporating virtual scenarios which you think may prepare the learner for real-world scenarios.
